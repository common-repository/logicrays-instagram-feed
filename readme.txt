== Description ==
Display Instagram Feeds from your account, This plugin works with single feed from one account.

== Features==
Very easy to <strong>configure.</strong>
Responsive and <strong>mobile friendly.</strong>
Use the built-in <strong>shortcode.</strong>
Select columns layout, two columns , three columns etc.</strong>
Set limit for images. per page 10,15
Set View all links to redirected to your <strong>instagram page.</strong>
View images by <strong>slider(OWl Carousal)</strong>
For more features Please contact us at [Logicrays](http://www.logicrays.com/contact-us-logicrays-india/)

== Installation ==

1. Upload Zip file lr-instagram to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php echo do_shortcode('[LR_INSTAGRAM_FEED] ');?>` in your templates files or You can add [LR_INSTAGRAM_FEED] to post,pages.
